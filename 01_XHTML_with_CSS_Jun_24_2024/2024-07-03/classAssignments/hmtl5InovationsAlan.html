10 Extensions for the VS Code

  1. Close Tags
    Tag elements are automatically closed when the > of the opening tag is typed.
  2. Auto Update Tags
    When modifying a tag, the linked editing feature automatically updates the matching closing tag. This feature is optional and can be enabled via settings.
  3. Color Picker
    The VS Code color picker UI is now available in HTML style sections.
  4. Hover
    Move the mouse over HTML tags or embedded styles and JavaScript to get more information about the symbol under the cursor.
  5. Emmet Snippets
    VS Code supports Emmet snippet expansion, listing Emmet abbreviations along with other suggestions and snippets in the editor auto-completion list.
  6. Debugger for Chrome
    This extension is used for debugging JavaScript code in the Google Chrome browser within the Visual Studio Code environment. It offers a more convenient debugging experience compared to the Chrome console. To use it, install the extension and set up the launch.json settings for the webpage you want to debug.
  7. Live Server
    This extension launches a local development server with a live reload feature for static and dynamic pages, automatically refreshing the browser to reflect changes made in VS Code when you save the file.
  8. Beautify
    This extension formats files like HTML, converting unformatted code into readable, well-structured code. You can set this preference in VS Code to automatically format code as you type.
  9. Material Icon Theme
    This extension provides specific and colorful icons for your project files based on their extensions, making it easier to identify them. This is a popular icons theme for many users.
  10. 19 Excel Viewer
    View Excel files in VS Code. Many of us have likely spent considerable time working with Excel spreadsheets. Even if you're a Python enthusiast using the pandas library, you've probably encountered Excel files at various points. This VS Code extension is a fantastic tool as it allows you to preview Excel files directly in your code editor. Staying within your coding environment when handling different file types can be a significant time saver. Whether you need to work with Excel spreadsheets or CSV files, this extension lets you quickly preview data without needing to open another application.
